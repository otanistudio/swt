Stupid Web Tricks: Naïve Random Name Generator

Very simple. C = Consonant and V = Vowel.

It will use one of the two following patterns to generate a random name: cvcc(v|c|v) or vccv(c|v|c). Hit "generate" to create a new set of a dozen random names.

I had done this for some other dude really quick during a hack week, and I wrote the JS version over a year ago after reading about the simple consonant-vowel algorithm on some dude's blog, but I can't find it. I'd like to cite it and give credit, but honestly... I can't remember where I read it, but it's probably a pretty common entry-level textbooky algorithm.
