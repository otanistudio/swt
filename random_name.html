Stupid Web Tricks: Naïve Random Name Generator

Very simple. C = Consonant and V = Vowel.

It will use one of the two following patterns to generate a random name: cvcc(v|c|v) or vccv(c|v|c). Hit "generate" to create a new set of a dozen random names.

Source: https://github.com/otanistudio/swt/blob/master/random_name.html
